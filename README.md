## 快速开始

🔍 打开微信，搜索小程序「苯苯存图」，无需进行任何配置即可使用。

## 支持功能

✅ 输入单个或多个链接，批量保存小红书图片或视频。

✅ 输入单个或多个链接，批量保存米游社图片。

✅ 输入单个或多个链接，批量保存微博图片。

## 支持功能

✅ 输入单个或多个链接，批量保存小红书 **无水印** 图片或视频。

✅ 输入单个或多个链接，批量保存米游社图片。

✅ 输入单个或多个链接，批量保存微博图片。

## 预计支持

🚧 输入单个或多个链接，批量保存 Lofter 图片。

🚧 输入单个或多个链接，批量保存抖音视频。

🚧 输入单个或多个链接，批量保存哔哩哔哩视频。
